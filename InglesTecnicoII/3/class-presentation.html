Developer Glosary

Pablo Sanchez Galdamez (21001135)

14/10/21

API

Definition

Application Program Interface

Examples

  1. The API that we are using will be deprecated, its time to update our code.
  2. The public API of your libraries should be as minimal as possible.

Browser

Definition

The program you use to access the Web — such as Chrome, Firefox, or Safari.

Examples

  1. Go to your browser, and visit 192.168.0.1, there you will be able to change the password of your WiFi.
  2. In web dev, we need to take into account all web browsers.

Cache

Definition

The storage of certain elements to help with faster load times.

Examples

  1. Delete the cache and try again, that usually solves the problem.
  2. Thanks to the cache, the website can load faster.

DevOps

Definition

“Development Operations.” System of working that helps to keep development, IT operations, and quality assurance departments on the same page to make for better end-products and collaborations.

Examples

  1. I’m an expert int DevOps.
  2. The Incremental model can be applied to DevOps.

Framework

Definition

Suite of programs used in website or software development. This lays the groundwork for the type of programming language used for your site or app development.

Examples

  1. You don’t need to learn all the new frameworks.
  2. It seems like every day there is a new and hot JS framework.

GUI

Definition

Graphical user interface.

Examples

  1. You don’t need a GUI to use a computer, the command line is good enough.
  2. A GUI will make your app user friendly.

HTML

Definition

Hypertext Markup Language

Examples

  1. HTML is not a programming language.
  2. HTML alone is ugly, you need CSS.

OOTB

Definition

Out of the box. These are the ready-made, plug-and-play options for features and functions that you can download and install without the need to customize or configure.

Examples

  1. A USB works OOTB.
  2. Your apps should work OOTB, if you need your user to set it up and configure it manually, you are doing it wrong.

Server

Definition

Computers running software that allows users to access your website — this is what houses the hosting. If your domain is the website’s mailing address and the hosting is its house, the server is the land it’s built on.

Examples

  1. For some reason the server is down.
  2. We will need a new server, the website is getting too much traffic.

404

Definition

Error message when what was requested cannot be found; often because the link that was requested is either broken or dead.

Examples

  1. The site always returns a 404, I think the link is broken.
  2. I hate when an API returns 200 even if something failed. No 500, no 404, no 501, always 200.
